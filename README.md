## 📖 What is llm-chat?

llm-chat is a simple tool that helps manage conversations with large language models (LLMs). It keeps track of conversation history, system instructions, and the length limits of the interaction. You don’t need to install extra software or use any complicated tools. It works right away on your Windows PC.

This tool is written in C++ and made for easily handling chat sessions with AI APIs. It is designed to keep conversations organized and smooth without the need to learn programming.
